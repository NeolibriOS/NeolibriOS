# NeolibriOS

WIP: switch from Tup to Meson, drop extra programs and ports

[![License](https://img.shields.io/badge/License-GPL%202.0-green)](./COPYING.TXT)

NeolibriOS is tiny and fast hobby operating system for x86-compatible computers, based on KolibriOS.

The project focused on fixing fundamental problems of KolibriOS and adding new important features in a more free way, without looking back at many outdated legacy stuff

Its kernel, most drivers, and many programs are written entirely in [FASM](https://flatassembler.net/) assembly language, making them very compact and lean on system resources.

KolibriOS itself (of which NeolibriOS is a fork), is based on [MenuetOS](https://www.menuetos.net/), it uses its own standards and is NOT fully POSIX or UNIX compliant.
